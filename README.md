## 🎯 Sobre o Projeto

O **He-Art Application** é um projeto focado em **[No desenvolvimento de uma rede social direcionada a artistas, onde é criado uma comunidade forte e comp-anheira]**. 

O foco principal deste repositório foi construir uma aplicação fluida, focando bastante na experiência do usuário (UX) e em uma interface visualmente agradável (UI).

## 🛠️ Tecnologias Utilizadas

* **[Linguagem]** PHP
* **[Estilização]** CSS
